四、连接git的准备

  • 命令行执行ssh-keygen -t rsa -C “邮件地址@youremail.com”在本地生成新的 SSH Key
  • 在本地文件夹找到id_rsa.pub文件并打开复制里面的内容,进入GitHub主页，进入settings，左侧选SSH and GPG keys，将复制的内容粘贴到对应位置，title随便填个
  • 命令行执行ssh-T git@github.com提示图示2即为成功
  • 命令行执行git config –global user.name “username”//输入注册时的username
  • 命令行执行git config –global user.email “yourname@yourmail.com“//填写注册邮箱
  • 创建GitHub仓库，仓库必须为username.github.io
